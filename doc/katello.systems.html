katello.systems documentation

add-to-system-group
(add-to-system-group system-group system-name)
Adds a system to a System-Group

copy-system-group
(copy-system-group orig-name new-name & [{:keys [description]}])
Clones a system group, given the name of the original system group
to clone, and the new name and description.
create-system
(create-system name & [{:keys [sockets system-arch]}])
Creates a system

create-system-group
(create-system-group name & [{:keys [description]}])
Creates a system-group

edit-system
(edit-system name {:keys [new-name description location release-version]})
Edits the properties of the given system. Optionally specify a new
name, a new description, and a new location.
edit-system-group
(edit-system-group sg-name {:keys [new-limit new-sg-name description]})
Change the value of limit field in system group

subscribe-system
(subscribe-system {:keys [system-name add-products remove-products auto-subscribe sla]})
Subscribes the given system to the products. (products should be a
list). Can also set the auto-subscribe for a particular SLA.
auto-subscribe must be either true or false to select a new setting
for auto-subscribe and SLA. If auto-subscribe is nil, no changes
will be made.
